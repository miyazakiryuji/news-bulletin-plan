# ストーリー（構成案）

「ニュース速報」風に、新郎・**大川こうき** さんの結婚を真顔で報道していく構成。
メインは**ニュースのスタジオ画角**。速報から**現地インタビュー**へとつなぎ、
クライマックスで**式場と“生中継”をつないで本物の新郎新婦を巻き込む**のが最大の仕掛け。

- 新郎：大川こうき　／　新婦：ちさと
- リポーター（現地）：大ちゃん（水川大地）
- 差し込み画像は **二人が写っているもの** を使う
- ※タイムコードは目安。生パート（現地中継）は当日の尺で前後する → 進行の詳細は `当日進行.md`
- ※コーナー企画（経歴VTR／関係者証言）は一旦ストーリーから外して `意見.md` に保留中

## 0:00 冒頭テロップ（免責ギャグ）

- 黒画面に注意書き風テロップ：
  > 「こうきの知人によって作成された動画のため、一部情報にかたよりがございます。」
- “身内が作った偏った速報” という前提を先に共有して、笑いの免罪符にする

## 0:00–0:30 速報スタート

- 「ニュース速報」テロップ＋緊急ジングルでドン！（ニュースの画角・「スクープです」のテンション）
- 畳みかけの速報。**②③は画面に割り込む「帯（テロップ帯）」で出す**：
  1. **速報** … 何かが起きた気配
  2. 【帯】**ウェザーニュース風**「大川の心の晴れ模様」（気象予報士役）
  3. 【帯】**緊急地震速報パロディ**「大川の自信」… 自信満々の顔写真を割り込み表示

## 0:30–3:00 電撃結婚 速報 → 現地インタビュー

- キャスター真顔で：「速報が入りました。**こうきさんと、ちさとさんが電撃結婚** をしたとの速報が入りました。」
- 画面下に **ティッカー（速報コメント）** … 祝福・野次＋時々ボケ＋“誤報訂正”を流し続ける
- 「現地はお祝いムードです」→ **現地インタビュー（街頭インタビュー形式）** に切替（街頭の **特派員** は録画。式場の生リポーター大ちゃんとは別人）
  - スタジオから街角の記者へ。背後の通行人は素通りなのに **特派員だけ妙に緊張**：「現場に来ております。ご覧のとおり、現場は…**とくに、いつもどおりです**」
  - 通行人・関係者・友人に次々マイクを向け、**テンポよく数珠つなぎ**（祝福→困惑→暴露→ほっこりと緩急をつけて飽きさせない）

**▼ 現地インタビューの流れ（0:30–3:00）**

- **〔0:30–0:52〕現場テンプレ（祝福の入口）**
  - 特派員「一帯はかつてないお祝いムードに包まれて…」と言った直後、背後の通行人は誰も気にせず素通り →「…包まれて、**おります**」と言い張る
  - ティッカー訂正〔先ほど『祝福ムード』とお伝えしましたが、正しくは『通常ムード』でした。お詫びして訂正します〕
- **〔0:52–1:18〕街の祝福と困惑（既存ネタの掴み）**
  - 通行人A（即答・満面の笑み）「最高です。」だけ言って去る → 特派員「…以上、最高だそうです。」
  - 通行人B（しみじみ）「いやぁ、まさか結婚するとは」→「お知り合いで？」→「いえ、**初めて知りました今**。」〔テロップ:本件を5秒前に知った人〕
  - 中年男性「日本中が待ってたんじゃないかな」（大げさ）〔テロップ:大川さんを知らない人(54)〕
- **〔1:18–1:40〕マイクトラブル（カオスの小休止）**
  - 話し始めるが音が出ない →マイクを叩くと途中から「————ッテ感じですよね！？」と急復活、特派員ビクッ〔ティッカー:ただいま音声トラブル／そのうち直ります〕
  - 去ったはずの通行人Aが、フレーム端でまだ手を振っている（こぼれギャグ）
- **〔1:40–2:18〕関係者の暴露ゾーン（顔モザイク＋音声加工。※旧「関係者証言」を吸収）**
  - 〔BGMシリアス転調・画暗め〕特派員「彼を古くから知る関係者は、複雑な思いも…」
  - 中学の同級生〔フルモザイク＋低音加工〕「こいつが結婚するなんて誰も思ってなかった。**将来の夢が“無職で生きていく”だった**ので」
  - 職場関係者〔モザイクさらに濃く〕「“締切を守れない男”として有名で…」→「式は…たぶん、間に合うと思います」（自信なさげ）※“重要人物ほど厳重モザイク”でエスカレート
  - メタ・ナレ「なお、証言の多くは“こいつ”呼びでお送りしております」〔割り込み帯:※全員、悪気はありません〕
- **〔2:18–2:42〕急ブレーキ＝実はいい話（転調）**
  - 最重要モザイクが一転、優しい声で「でも、ね。そんなあいつを“いい”って言ってくれる人が現れた。それが一番の奇跡」〔BGMあたたかい曲へ〕
  - モザイクがふっと外れて素顔・照れ笑い「**ちさとさん、ほんとにありがとう。こいつをよろしくお願いします**」〔テロップ:※モザイクは本人の照れ隠しでした〕
- **〔2:42–3:00〕ほっこり着地 → 生中継への振り**
  - 素顔の友人「“まさか結婚するとは”ってさっきの人も言ってたけど（笑）。でも、ほんと、おめでとう。**ちさとさんも素敵な人で**、幸せになってほしい」
  - 特派員「現場からは以上です。ほぼ暴露でしたが、温かい声が…たまに変な声も、届いておりました。スタジオにお返しします」と一礼
  - ティッカー〔このあと 式場と“生中継”／“大ちゃん”が式場へ／新郎、まさかの登場へ…〕→ そのまま次の **「水川さーん！」（3:00）** へ
