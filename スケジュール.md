# スケジュール

役割分担を含めて **6月後半まで** に企画を固める。
**式当日は10月10日**。会場へは動画を焼いたCDを当日の2週間前（9/26）までに渡す必要があり、それだとギリギリなので **できれば3週間前（9/19）までに納品** して段取りを固めておきたい。

| 時期 | フェーズ | 内容 |
| --- | --- | --- |
| 〜6月後半 | 企画・役割分担 | 構成と各メンバーの役割を確定 |
| 7月 | 撮影 | 各パートを撮影 |
| 8〜9月 | 編集 | 編集・仕上げ |
| 〜9/19（目標） | 納品 | CDに焼いて会場へ。段取り確認の余裕を持たせる |
| 9/26（締切） | 納品リミット | 式の2週間前。ここを過ぎるとアウト |
| 10/10 | 式当日 | 本番 |

## 納品の逆算メモ

- **10/10** … 式当日
- **9/26** … 会場への納品リミット（2週間前）。これがギリギリ
- **9/19** … 目標納品日（3週間前）。ここで渡して段取りを固めたい
- → 編集は **9月中旬には完了** させておきたい
