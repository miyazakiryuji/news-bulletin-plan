02
STORY ／ 構成案

ストーリー ── 約5分の報道タイムライン

「ニュース速報」風に、新郎・大川こうき さんの結婚を真顔で報道する 約5分 の構成。メインはニュースのスタジオ画角。速報から 現地インタビュー へとつなぎ、クライマックスで 式場と“生中継”をつなぎ本物の新郎新婦を巻き込む のが最大の仕掛け。録画コーナー（経歴VTR／関係者証言）は一旦 04 意見・アイデアメモ に保留中。生パートの段取りは 03 当日進行 へ。

REC 0:00 冒頭テロップ

免責ギャグ

  • 黒画面に注意書き風テロップ：「こうきの知人によって作成された動画のため、一部情報にかたよりがございます。」
  • “偏った身内速報” という前提を先に共有して笑いの免罪符に
  • 差し込み画像は 二人が写っているもの を使う
REC 0:00–0:30 PART 1

速報スタート

  • 「ニュース速報」テロップ＋緊急ジングルでドン！（「スクープです」のテンション）
  • 畳みかけの速報（②③は画面に割り込む 「帯」 で出す）：
  • ① 速報 … 何かが起きた気配
  • ②【帯】ウェザーニュース風「大川の心の晴れ模様」（気象予報士役）
  • ③【帯】緊急地震速報パロディ「大川の自信」… 自信満々の顔写真を割り込み表示
REC 0:30–3:00 PART 2

電撃結婚 速報 → 現地インタビュー

  • キャスター真顔で「速報が入りました。こうきさんと、ちさとさんが電撃結婚 をしたとの速報です」
  • 画面下の ティッカー … 祝福・野次＋時々ボケ＋“誤報訂正”を流し続ける
  • 「現地はお祝いムードです」→ 現地インタビュー（街頭インタビュー形式） に切替。街頭の特派員は録画（式場の生リポーター大ちゃんとは別人）。背後の通行人は素通りなのに特派員だけ妙に緊張：「現場は…とくに、いつもどおりです」
  • 通行人・関係者・友人に次々マイクを向け テンポよく数珠つなぎ（祝福→困惑→暴露→ほっこりの緩急）
🎤現地インタビューの流れ（0:30–3:00）
  • 〔0:30–0:52〕現場テンプレ：「お祝いムードに包まれて…」直後に通行人が素通り →「…包まれて、おります」と言い張る／ティッカー訂正〔『祝福ムード』→正しくは『通常ムード』でした〕
  • 〔0:52–1:18〕街の祝福と困惑：通行人A「最高です。」だけで去る →「…以上、最高だそうです。」／B「まさか結婚するとは」→「お知り合いで？」→「いえ、初めて知りました今」〔本件を5秒前に知った人〕／〔大川さんを知らない人(54)〕
  • 〔1:18–1:40〕マイクトラブル：音が出ない →叩くと「————ッテ感じですよね！？」と急復活、特派員ビクッ／去った通行人Aがフレーム端でまだ手を振る
  • 〔1:40–2:18〕関係者の暴露（顔モザイク＋音声加工・旧「関係者証言」を吸収）：中学同級生「将来の夢が“無職で生きていく”だった」／職場「“締切を守れない男”…式は、たぶん間に合うと」※重要人物ほど厳重モザイク／メタ帯〔※全員、悪気はありません〕
  • 〔2:18–2:42〕急ブレーキ＝実はいい話：最重要モザイクが一転優しく「そんなあいつを“いい”って言ってくれる人が現れた、それが奇跡」→モザイクが外れ素顔「ちさとさん、ほんとにありがとう」〔※モザイクは照れ隠しでした〕
  • 〔2:42–3:00〕ほっこり着地 → 生中継へ：素顔の友人「おめでとう。ちさとさんも素敵な人で、幸せになってほしい」→特派員「現場からは以上です。スタジオにお返しします」→ そのまま 「水川さーん！」(3:00) へ

03
RUN SHEET ／ 当日進行

当日進行 ── 生パート（参戦ポイント）

動画の途中で 一旦“生中継”に切り替え、本物の新郎新婦を巻き込む のが最大の仕掛け。ここは録画ではなく 式当日その場で実演 する。

🔀切り替えイメージ
  1. 動画を再生（冒頭 〜 現地インタビュー）
  2. 動画内でスタジオが「水川さーん！」と振る ＝ 生パート開始の合図
  3. 大ちゃん（水川）が会場で生登場 → 新郎新婦にインタビュー
  4. こうき が生プロポーズ → ちさと「まずはありがとう」→ 大ちゃんが一言（感想）でスタジオに戻す
  5. 暗転 → 動画の続き（締め 〜 オチ 〜 熱盛）を再生

▼ ポイント：動画は「水川さーん！」直後で 一旦区切れる作り（前半・後半の2チャプター） にしておくと当日オペが楽。生パート（現地中継）に 約1分30秒間を確保（当日ぐだる可能性を見込んだバッファ）。

役割当日の役割分担

生パートの担当（「要決定」は埋めていく）
役割担当やること
映像オペ（再生）要決定前半再生 → 生パートで停止 → 暗転後に後半再生
現地リポーター大ちゃん（水川大地）「水川さーん！」を受けて生登場、新郎新婦に振る
マイク手配要決定新郎・新婦・リポーター用のワイヤレスマイク
照明・暗転会場／要決定「暗転」の合図出し → 動画再開のタイミング
新郎フォロー要決定こうき に直前で段取りを耳打ち

仕込み新郎新婦への仕込み（要検討）

  • 新郎（こうき）：プロポーズの言葉をもう一度。事前に軽く伝えておくと安心。
  • 新婦（ちさと）：「まずはありがとう」を本人に言わせるのが最大の見せ場。
    • A案：本人に伏せてサプライズ（リアクション重視・ただし出ない可能性）
    • B案：大ちゃんが「あの時なんて返しましたっけ？」と誘導（安全）→ B案推奨

確認会場への事前確認

  • スクリーン投影＋音声、暗転（照明操作）が可能か
  • ワイヤレスマイクを余興で使えるか／本数
  • 動画を2チャプターに分けて渡してよいか（前半・後半）
